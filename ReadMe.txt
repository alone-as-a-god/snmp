Es gibt zwei Möglichkeiten mein Programm zu starten:

1. im "dist" Ordner, die main.exe ausführen
2. Python installieren. Hierbei wird Python 3.8 benötigt. Daraufhin in der Konsole mit "pip install" folgende pakete installieren:
 - keyboard
 - pysnmp
Daraufhin die main.py starten. Programm sollte voll funktionsfähig sein

Zum verwenden des Programms, nach Aufforderung "/help" eingeben (ohne anführungszeichen)
